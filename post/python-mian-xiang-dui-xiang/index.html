Python面向对象
Python面向对象
  • 1.opp
    • 在完成某一个需求前，首先确定 职责 —— 要做的事情（方法）
    • 根据 职责 确定不同的 对象，在 对象 内部封装不同的 方法（多个）
    • 最后完成的代码，就是顺序地让 不同的对象 调用 不同的方法
  • 2.类
    • 就相当于制造飞机时的图纸，是一个 模板，是负责创建对象的
    • 类 是对一群具有 相同 特征 或者 行为 的事物的一个统称，是抽象的，不能直接使用
    • 特征 被称为 属性
    • 行为 被称为 方法
    • 它也是一种分类方式，比如说一类事物——鸟
  • 3.对象
    • 对象是由类创建出来的一个具体存在，由 哪一个类 创建出来的 对象，就拥有在 哪一个类 中定义的：
    • 属性
    • 方法
    • 对象 就相当于用 图纸 制造 的飞机
    • 在程序开发中，应该 先有类，再有对象
  • 4.__方法名__ 
    • 是 Python 提供的 内置方法 / 属性
    • __init__
      • 改造初始化方法，可以让创建对象更加灵活
    • __del__ 
      • 如果希望在对象被销毁前，再做一些事情，可以考虑一下 __del__ 方法
    • __str__ 
      • 希望使用 print 输出 对象变量 时，能够打印 自定义的内容，就可以利用 __str__ 这个内置方法了
  • 5.面向对象的三大特性
    • 1.封装
      • 封装 是面向对象编程的一大特点
      • 面向对象编程的 第一步 —— 将 属性 和 方法 封装 到一个抽象的 类 中
      • 外界 使用 类 创建 对象，然后 让对象调用方法
      • 对象方法的细节 都被 封装 在 类的内部
      • 例子
        • 人是属于一个类，郭新海就是人定义的对象
    • 2.继承
      • 实现代码的重用，相同的代码不需要重复的编写
      • 例子
        • 单继承：儿子继承父亲的基因
        • 多继承：儿子可以继承父亲和母亲的基因
    • 3.多态
      • 不同的对象调用相同的方法，产生不同的执行结果，增加代码的灵活度
  • 6.私有属性
    • 对象 不希望公开的 属性
    • 私有属性、方法 是对象的隐私，不对外公开，外界 以及 子类 都不能直接访问
  • 7.私有方法
    • 对象 不希望公开的 方法
    • 私有属性、方法 是对象的隐私，不对外公开，外界 以及 子类 都不能直接访问
  • 8.方法的重写
    • 子类的方法不能满足需求
    • 子类可以写与父类相同的方法
    • 如果要输出父类的方法（super.method())
  • 9.实例
    • 使用面相对象开发，第 1 步 是设计 类
    • 使用 类名() 创建对象，创建对象 的动作有两步：
      • 1) 在内存中为对象 分配空间
      • 2) 调用初始化方法 __init__ 为 对象初始化
    • 对象创建后，内存 中就有了一个对象的 实实在在 的存在 —— 实例
    • 创建对象的 动作 叫做 实例化
    • 对象的属性 叫做 实例属性
    • 对象调用的方法 叫做 实例方法
